Troubleshooting

Currently, SlenderTubbies shouldn't have major issues with launching.

If you are the installing game from the Itch app, you shouldn't have any redistributable issues. If this is your first time running an Unreal Engine game, the game should automatically open the engine redistruables setup. The same applies to .Net games as well.

If you have any trouble launching the game, please visit the Misiu Games Studio Discord server. See you there!
