Troubleshooting

Currently, SlenderTubbies shouldn't have major issues with launching.

If you are installing the game from the Itch app, you shouldn't encounter any redistributable issues. If this is your first time running an Unreal Engine game, the game should automatically run the engine redistributable setup. The same applies to .Net games as well.

If you have any trouble launching the game, please visit the Misiu Games Studio Discord server. See you there!
