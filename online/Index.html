How can I play SlenderTubbies online?

This document will help you join multiplayer games and manage connected players.

First, let's talk about how to join a server.

Before we start, make sure that:

  • You are logged in to Epic Games.
  • You have allowed the game to bypass Windows Firewall.
  • You are using the same game version.
  • You have a stable Internet connection.

Press the Refresh button at the top of the menu and wait for your server to appear on the Server List.

Hosting an online server is the same as mentioned in Match Creation, the only change is to set the Player Count option to a value greater than 1.

If you still can't play with your friends, feel free to contact me on the Misiu Games Studio Discord server.

Keep in mind that I'm not active 24/7. Thank you. :)
