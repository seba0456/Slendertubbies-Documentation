How do I create a match in SlenderTubbies?

This document discusses all the steps of creating a match in SlenderTubbies.

Firstly, in the main menu, you have to press the "Create" button. For now, we are going to skip networking elements as I explain them later in the "Online Play" category. By default, the created match will be Solo.

  • Game Modes

    SlenderTubbies has 2 different game modes. Both game modes are available in Singleplayer and Multiplayer.

    • Collect

      This is a classic horror game mode. The player must collect all the custards if they wish to win the game. The player takes the role of the victim, with the time to collect the custards being unlimited. If you are caught by a Slendertubby, you will lose.

    • Versus

      This game mode is a bit more complicated. The host player takes on the role of the Slendertubby, their goal is to eliminate all survivors before they can collect all custards. All other players are survivors who must collect all the custards and not let the Slendertubby catch them. The Slendertubby will win when all players are caught. Survivors will win when all custards are collected. This is mainly a Multiplayer game mode that works best with real players. However, you can play it with bots by pressing 'M' in-game if needed.

  • Maps

    SlenderTubbies has 10 different maps with an additional bonus map you can unlock. All of them are available in Singleplayer and Multiplayer, with each map name essentially describing what the specified map will contain.

    • Woods
    • Woods (Legacy)
    • Maze
    • Lake
    • Bunker
    • Camp
    • Fortress
    • Wastelands
    • Warehouse
    • Riverside

    My personal favorite maps are Bunker, Wastelands, Fortress and Maze.

  • Custard Amount

    This is pretty self-explanatory. Using the slider, you can specify how many custards will spawn.

    The default amount of custards is 10.

  • Toggles

    Toggles allow for the given match to include some unique elements. As you see, the game has only 3 of them. They can make the match harder or more unique depending on which toggles you enable.

    • Select Slendertubby

      Typically, all maps have a pre-determined Slendertubby. This toggle allows you to select which Slendertubby will spawn. In the Versus game mode, this toggle will allow the host to select which Slendertubby they will be. Maze has this option always enabled.

    • Battery

      This toggle adds power consumption to your flashlight. When active, the player's flashlight loses power slowly. To keep your light from going out, you must collect batteries found around the chosen map.

      This toggle functions in both Singleplayer and Multiplayer.

    • Full Tubby Jacket

      This rule spawns every possible Slendertubby into the match. This toggle makes the game really challenging.

      This toggle works only in the Collect game mode. This toggle works in both Singleplayer and Multiplayer.

  • Create

    This button starts the game.
